DePaulCSC459Project

To run the application:
From Command Line:
java -cp ".;.\RobotSystem-0.0.1-SNAPSHOT.jar;.\SensorSystem-0.0.1-SNAPSHOT.jar" src.main.java.CleanSweepModels.HomeUI
